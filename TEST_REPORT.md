## 四、发现的问题

### P0 - 无

### P1 - 高优先级
1. **GitHub API 限流**: 在没有配置 GITHUB_TOKEN 的情况下，API 请求容易触达速率限制，导致搜索返回空结果。
   - **建议**: 用户应配置个人 GitHub Token 以提高 API 配额。

### P2 - 中优先级
1. **数据库初始化延迟**: 首次启动时数据库初始化和表创建耗时约 3 秒。
   - **建议**: 可考虑使用预生成的 SQLite 文件来减少首次启动时间。

### P3 - 低优先级
1. **主题切换首次响应略慢**: 第一次主题切换耗时 156ms，后续稳定在 125ms 左右。
   - **原因**: CSS 过渡动画首次渲染开销。
   - **影响**: 用户几乎不可感知。
